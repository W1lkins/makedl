# makedl

Fetch `Makefile` files from Github.

## Installation

#### Via Go

```console
$ go get -u -v github.com/w1lkins/makedl
```

## Usage

```console
$ makedl [language]
```

This will download `[language].Makefile` from https://github.com/W1lkins/makefiles
to the current working directory.
